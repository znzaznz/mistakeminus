# 财务管理 知识点要义 · 第8-8章

## 成本管理的目标 `掌握`  
- 指企业通过成本预测、决策、计划、控制、核算、分析和考核等手段，实现成本最小化与效益最大化的统一，核心是“在保证质量与功能前提下持续降低单位产品/服务的资源耗费”。  
- 包含三重目标：成本水平控制目标（如单位成本下降率）、成本结构优化目标（如制造费用占比压降）、战略协同目标（如支持差异化或低成本战略）。  
- 常考易错点：混淆“成本最低”与“成本最优”——成本管理不追求绝对最低（可能牺牲质量或交付），而追求相关范围内总成本（含隐性成本）最小；不得将目标简单等同于削减人工或材料费。  
- 命题敏感点：题干常设置“为达成某战略（如快速抢占市场）而主动提高某环节成本”的情境，考查对目标动态性与战略适配性的理解。

## 成本管理的原则 `熟悉`  
- 包括成本效益原则（支出带来的收益≥成本）、融合性原则（成本管理嵌入业务流程而非财务孤岛）、适应性原则（方法匹配企业规模、行业特征与管理基础）、重要性原则（聚焦关键成本动因与重大差异项目）。  
- “融合性”强调业财融合：如生产部门参与标准工时制定、采购部门协同确定材料价格标准，非仅财务部单方定标。  
- 常考易错点：误将“全面性原则”列为正式原则（教材未列）；混淆“适应性”与“一贯性”——适应性允许不同车间采用不同成本核算方法（如机加工用作业成本法、装配用品种法）。  
- 命题敏感点：选项常以“所有部门必须统一使用标准成本法”“成本分析只需财务人员完成”等表述设置干扰，考查对原则内涵的辨析。

## 成本管理的主要内容 `掌握`  
- 包含六项：成本预测（基于历史数据与经营计划估算未来成本）、成本决策（如自制/外购、是否接受低价追加订单）、成本计划（编制成本预算及分解指标）、成本控制（执行中偏差监控与纠偏）、成本核算（按对象归集分配，区分完全成本法与变动成本法）、成本分析与考核（差异归因、责任追溯、绩效挂钩）。  
- 各环节逻辑闭环：成本预测→支撑成本决策→形成成本计划→驱动成本控制→依赖成本核算提供数据→通过成本分析反馈优化预测模型。  
- 常考易错点：遗漏“成本决策”独立地位（易误并入“成本计划”）；混淆“成本控制”与“成本核算”时点——控制是事中行为，核算是事后计量。  
- 命题敏感点：案例题常给出某企业仅做成本核算但无差异分析、无考核挂钩，问缺失哪一环节，需精准对应“成本分析与考核”。

## 本量利分析的含义 `掌握`  
- 指研究成本、业务量（产量或销量）、利润三者之间数量依存关系的方法，本质是建立线性数学模型（y = a + bx）描述短期经营成果变动规律。  
- 核心变量：固定成本（a）、单位变动成本（b）、单价（p）、销量（x）、利润（P），基本公式为 P = (p - b)x - a。  
- 常考易错点：“本量利”中“本”特指**成本性态分类后的固定成本与变动成本**，非全部实际成本（如混合成本须先分解）；不适用于长期决策（因固定成本会随规模改变）。  
- 命题敏感点：题干若出现“折旧费随产量增加而上升”“管理人员工资按销售额提成”等表述，需先识别成本性态错误，再判断能否直接套用本量利模型。

## 本量利分析的基本假设 `熟悉`  
- 四项刚性前提：相关范围内成本性态保持线性（固定成本总额不变、单位变动成本不变）、产销平衡（产量=销量，无存货变动）、产品产销结构稳定（多产品时各产品收入占比恒定）、销售价格不变（不考虑折扣、批量价等）。  
- “相关范围”是关键限定：超出该范围（如产能翻倍需新建厂房），固定成本台阶式上升，原模型失效。  
- 常考易错点：误认为“假设单位变动成本不变”即忽略技术进步导致的单耗下降；混淆“产销平衡”与“零存货”——允许期初/期末存货存在，但要求其金额相等（变动成本法下利润不受存货影响）。  
- 命题敏感点：选项常以“企业正进行大规模技改，单位材料消耗显著降低”“产品组合由A:B=3:1变为1:2”为背景，考查假设被突破的情形。

## 本量利分析的基本原理 `掌握`  
- 基于边际贡献概念：单位边际贡献 = p - b，边际贡献总额 = (p - b)x，用于补偿固定成本后形成利润，即 P = 边际贡献总额 - a。  
- 盈亏平衡点销量 x₀ = a / (p - b)，盈亏平衡点销售额 S₀ = a / [(p - b)/p] = a / 边际贡献率。  
- 常考易错点：计算盈亏平衡点时误用“单位产品利润”（p - b - a/x）替代单位边际贡献；多产品下必须用加权平均边际贡献率，不可简单加总各产品x₀。  
- 命题敏感点：题干给出“安全边际率20%”“边际贡献率40%”，要求反推固定成本，需熟练运用 S₀ = 销售额 × (1 - 安全边际率) 及 a = S₀ × 边际贡献率。

## 本量利分析的优缺点 `熟悉`  
- 优点：模型简洁、直观反映成本结构影响、便于敏感性测试、支撑短期经营决策（如调价、增产临界点判断）。  
- 缺点：依赖线性假设导致长期决策失真；忽略风险因素（如需求不确定性）；多产品时加权处理掩盖单品风险；未考虑存货变动对利润的影响（与完全成本法结果差异）。  
- 常考易错点：将“优点”答为“适用于所有决策”（实则不适用资本支出决策）；混淆“缺点”与“假设”——如“需分解混合成本”是应用前提，非模型固有缺点。  
- 命题敏感点：问“某企业拟投资新生产线，能否用本量利分析评估？”正确答案为“否”，因涉及长期固定成本变动及投资回收期，超出模型适用边界。

## 盈亏平衡分析 `掌握`  
- 指计算使利润为零的业务量（盈亏平衡点）的过程，核心是求解 (p - b)x - a = 0，得出 x₀ = a / (p - b)。  
- 多产品企业需先计算综合边际贡献率：Σ(各产品销售收入 × 该产品边际贡献率) / 总销售收入，再得综合盈亏平衡销售额 = a / 综合边际贡献率。  
- 常考易错点：安全边际量 = 实际销量 - x₀，安全边际率 = 安全边际量 / 实际销量；二者与盈亏平衡点呈反向关系，非正比。  
- 命题敏感点：题干给出“实际销售额500万元，盈亏平衡销售额300万元”，直接可得安全边际率=40%，若问“固定成本增加50万元后安全边际率变化”，需重新计算新S₀。

## 目标利润分析 `掌握`  
- 在本量利模型中令 P = 目标利润，反推所需销量 x = (a + P) / (p - b) 或所需销售额 S = (a + P) / 边际贡献率。  
- 可用于倒推定价：p = b + (a + P)/x，或倒推成本控制目标：b = p - (a + P)/x。  
- 常考易错点：目标利润必须为税后利润时，需先换算为税前利润（税前=税后÷(1-所得税率)）再代入公式；多产品下目标利润需按销售比重分摊至各产品。  
- 命题敏感点：题干设“公司要求年净利润360万元（所得税率25%）”，则公式中P应取480万元（360÷0.75），此为高频扣分点。

## 敏感性分析 `掌握`  
- 指测算各参数（p、b、a、x）单位变动对利润影响程度，常用指标为敏感系数 = 利润变动百分比 / 参数变动百分比，系数绝对值越大越敏感。  
- 单位售价敏感系数 = 基期边际贡献 / 基期利润，固定成本敏感系数 = -基期固定成本 / 基期利润（负号表示反向变动）。  
- 常考易错点：敏感系数可正可负（如固定成本系数为负），但“敏感性排序”只看绝对值；销量敏感系数恒为正且通常最高，因同时影响收入与变动成本。  
- 命题敏感点：给出各参数敏感系数（如售价1.5、销量1.2、单位变动成本-1.0、固定成本-0.8），问“最需关注哪个风险”，答案为售价（绝对值最大）。

## 边际分析 `掌握`  
- 指基于边际贡献（MC = p - b）进行决策：当MC > 0时，增加一单位销量可增加利润（MC值）；当MC < 0时，减产可止损。  
- 关键应用：亏损产品是否停产——若该产品MC > 0，停产将减少总利润（因固定成本仍需承担）；特殊订单是否接受——只要MC > 0且不冲击正常产能，即可接受。  
- 常考易错点：“边际贡献为正”不等于“产品盈利”，因未覆盖固定成本；停产决策需比较“继续生产MC总额”与“停产释放资源创造的边际贡献”。  
- 命题敏感点：题干设“某产品单价100元，单位变动成本70元，固定成本50万元，销量1.2万件”，问“是否停产”，需计算MC=30元>0，故不应停产。

## 本量利分析在经营决策中的应用 `掌握`  
- 具体场景：①新产品投产临界销量测算；②亏损产品是否停产（MC是否＞0）；③是否接受特殊订单（MC＞0且产能剩余）；④调价决策（新价格下x₀是否低于预计销量）；⑤租赁/自购设备选择（比较不同固定成本结构下的x₀）。  
- 决策核心逻辑：所有应用均回归到“边际贡献能否覆盖专属固定成本”及“总利润是否提升”。  
- 常考易错点：特殊订单需剔除常规变动成本（如无销售佣金），但必须包含专属变动成本（如定制模具费）；多方案比较时，须统一计算各方案总利润，而非仅看单位MC。  
- 命题敏感点：题干设“客户愿以80元订购2000件（正常售价120元），单位变动成本65元，需新增专用设备租金5万元”，则新增MC=15×2000=3万元＜5万元，应拒绝。

## 标准成本的制定 `掌握`  
- 指在正常生产经营条件下，预先科学测定的单位产品应发生的成本，包括直接材料标准成本（标准用量×标准价格）、直接人工标准成本（标准工时×标准工资率）、制造费用标准成本（标准工时×标准制造费用分配率）。  
- “标准”分理想标准（无损耗、无故障）与正常标准（含合理损耗、机器维修时间），考试默认采用正常标准。  
- 常考易错点：材料标准价格不含增值税（为买价+运杂费-合理损耗抵扣），但含检验费；人工标准工资率含社保等附加成本，非仅基本工资。  
- 命题敏感点：题干给“材料发票价10元/公斤，运费0.5元/公斤，入库损耗率2%”，则标准价格= (10+0.5)/(1-2%)≈10.71元/公斤，损耗需分摊至有效用量。

## 成本差异的计算与分析 `掌握`  
- 直接材料差异：价格差异 = (实际价格 - 标准价格) × 实际用量；用量差异 = (实际用量 - 标准用量) × 标准价格。  
- 直接人工差异：工资率差异 = (实际工资率 - 标准工资率) × 实际工时；效率差异 = (实际工时 - 标准工时) × 标准工资率。  
- 制造费用差异：两差异法（耗费差异 + 能量差异）或三差异法（耗费 + 闲置能力 + 效率），考试以两差异法为主。  
- 常考易错点：用量/效率差异归属生产部门，价格/工资率差异归属采购/人事部门；能量差异 = (预算产量标准工时 - 实际产量标准工时) × 标准分配率，非实际工时。  
- 命题敏感点：题干给“实际产量1000件，实际工时2100小时，标准工时2小时/件，标准工资率15元/小时，实际工资率14.5元/小时”，则效率差异=(2100-2000)×15=1500元（不利）。

## 作业成本法的应用目标和应用程序 `熟悉`  
- 应用目标：①更精确分配间接费用（尤其间接费用占比高、产品多样性大时）；②识别增值/非增值作业，消除浪费；③支撑产品盈利分析与客户盈利能力管理。  
- 程序四步：①确认主要作业并划分作业中心；②归集作业成本（资源动因）；③确定成本动因（如订单数、检验次数）；④按动因分配作业成本至产品（作业动因）。  
- 常考易错点：“资源动因”连接资源与作业（如电费按机器小时归集到“设备维护”作业），“作业动因”连接作业与产品（如“订单处理”成本按订单数分摊）；不可混淆二者。  
- 命题敏感点：选项称“按生产工时分配全部制造费用”是传统成本法特征，若题干强调“质检次数多的产品分摊更多检验成本”，即体现作业成本法逻辑。

## 作业成本管理 `熟悉`  
- 指以作业成本信息为基础，通过作业分析、作业改进、作业消除、作业选择等手段提升价值创造效率的管理活动。  
- 核心是区分增值作业（顾客愿意付费，如核心加工）与非增值作业（不增加价值但暂时无法消除，如等待、返工），优先消除非增值作业。  
- 常考易错点：“作业成本管理”不等于“作业成本法”，前者是管理行为，后者是成本核算方法；作业成本法提供数据，作业成本管理是应用数据改进流程。  
- 命题敏感点：题干描述“企业发现‘材料搬运’作业占总成本25%但客户不感知价值”，问应采取措施，答案为“作业消除或作业减少”，非“提高搬运效率”。

## 责任中心及其考核 `熟悉`  
- 责任中心分四类：成本中心（只考核成本，如车间）、利润中心（考核收入与成本，如事业部）、投资中心（考核利润与投资额，如子公司）、收入中心（仅考核收入，较少见）。  
- 考核指标：成本中心用成本降低额/率；利润中心用边际贡献、可控边际贡献、部门税前利润；投资中心用投资报酬率（ROI = 部门利润/平均总资产）与剩余收益（RI = 部门利润 - 平均总资产×预期最低报酬率）。  
- 常考易错点：利润中心可控边际贡献 = 销售收入 - 变动成本 - 可控固定成本（不含公司分摊管理费）；投资中心RI避免ROI的短视倾向（如拒绝ROI低于当前但高于最低报酬率的项目）。  
- 命题敏感点：题干给“某利润中心销售收入200万元，变动成本120万元，可控固定成本30万元，不可控固定成本20万元”，则可控边际贡献=50万元，非30万元。

## 内部转移价格的制定 `了解`  
- 指企业内部责任中心之间提供产品或劳务的结算价格，类型包括：市场价格（有外部市场时首选）、协商价格（上下限为市价与单位变动成本）、双重价格（供方按市价、需方按成本）、成本转移价格（完全成本或变动成本，易导致目标不一致）。  
- 协商价格的上限为市价（需方不愿高于外购价），下限为供方单位变动成本（供方不愿低于增量成本）。  
- 常考易错点：“完全成本法转移价格”会导致供方缺乏成本控制动力（固定成本全转出）；“变动成本法”虽激励供方控本，但无法覆盖供方固定成本。  
- 命题敏感点：题干设“A部门向B部门提供零部件，外部市场价50元，A部门单位变动成本30元，单位固定成本10元”，则协商价格区间为30~50元，非40~50元。
