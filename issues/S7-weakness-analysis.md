# S7 — 薄弱点分析（知识点维度）

- **类型**：AFK
- **批次**：第二批
- **状态**：✅ 已完成
- **覆盖用户故事**：（衔接 V1 薄弱点需求，知识点维度）

## What to build

基于作答数据 + 知识点归属，给出按知识点维度的薄弱点分析。

- 后端纯函数：聚合每个知识点的做题数、正确率、错误次数、最近作答、掌握度，结合官方 `mastery_requirement` 权重，算出薄弱知识点排序。
- 分析维度：高频错知识点 / 正确率低的知识点 / 长时间未复习的知识点 / 重要程度高但掌握差的知识点。
- 接口 + 前端：薄弱点列表页，按知识点展示指标与建议复习优先级。

## Acceptance criteria

- [x] 后端聚合纯函数按知识点算出正确率/错误数/最近作答/掌握度
- [x] 结合能力要求权重输出薄弱知识点排序
- [x] 接口返回结构正确，接口级测试覆盖
- [x] 前端薄弱点页能按知识点展示指标与优先级
